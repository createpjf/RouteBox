Capabilities

Everything you need.
Nothing you don't.

A single app replaces multiple API dashboards, SDKs, and billing accounts.

Smart Routing

Automatically routes each request to the best model based on quality, speed, and cost. One endpoint, many models.

Cost Tracking

Real-time usage dashboard with per-model breakdowns, daily trends, and savings reports. Know exactly where your money goes.

OpenAI-Compatible API

Drop-in replacement for OpenAI. Change one URL and instantly access GPT-4o, Claude, Gemini, DeepSeek and more.

Native Desktop App

Built with Tauri for macOS. Runs a local gateway on your machine — your API keys never leave your device.

Cloud Mode

No API keys? Use RouteBox Cloud. Pre-pay credits, access all models instantly. Perfect for getting started fast.

Analytics & Insights

7-day trends, model comparisons, provider breakdowns. See what works and optimize your AI spending.

Setup

Three steps. That's it.

1

Download & Install

Install the macOS app. It sets up a local API gateway automatically.

2

Add Your API Keys

Paste your OpenAI, Anthropic, or Google keys. Or use Cloud mode with pre-paid credits.

3

Use One API Endpoint

Point your code to localhost:3001/v1. RouteBox handles the rest.
